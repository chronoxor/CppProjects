# How to clone, refresh and update?

## Clone all repositories

Windows
```
clone.bat
```

Linux
```
clone.sh
```

## Refresh all repositories

Windows
```
refresh.bat
```

Linux
```
refresh.sh
```

## Update all repositories

This step requires push access to repositories.

Windows
```
update.bat
```

Linux
```
update.sh
```
